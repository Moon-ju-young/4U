## 깃 브랜치

- `main`: 배포용, 최종 결과물이 올라가는 안정된 브랜치
- `dev`: 개발 통합 브랜치 (모든 기능 브랜치는 여기로 PR)
- `feat/*`: 기능 개발 브랜치 (이슈 기반으로 생성)
- `fix/*`: 버그 수정 브랜치 (필요 시 생성)

→ `dev` → 검토 & 테스트 완료 → `main`으로 머지

## 브랜치명 컨벤션

기능: feat/이슈번호-설명
버그: fix/이슈번호-설명
